## Remeash-App

### Prerequisites

* nvm (node version manager)
* NodeJS
* react-native cli
* Android SDK 
* Android SDK tools

### Installation 

* `$ nvm use`
* `$ npm install`

### Develop

* `$ react-native run-android`

### Build .apk 

* `$ cd android`
* `$ ./gradlew assembleRelease`

Generated APK can be found at:
`android/app/build/outputs/apk/release/app-release.apk`
